# rxjs

## 起動方法

```
npm run start
```

## サンプルの動かし方

```
<html>
  <head>
    <title>RxJS Fundamentals</title>
    <link href="styles.css" rel="stylesheet" />
  </head>
  <body>
    <p>scriptタグのjsを切り替えて使う。</p>
    <button type="button" id="btn">Click me!</button>

    <!-- scriptタグのjsを切り替える-->
    <script src="02_switchMap.js"></script>
  </body>
</html>
```
